Privacy Policy

Last updated: July 3, 2026

What this site collects

The calculators on ClearSum run entirely in your browser. The numbers you type in are never sent to, or stored on, any server. This site sets no cookies of its own and collects no personal information.

Hosting

ClearSum is hosted on GitHub Pages. GitHub may log standard technical data (such as IP addresses) for security and operational purposes, per the GitHub Privacy Statement.

Cookies & ads

ClearSum currently displays no advertising and uses no third-party cookies. If ads or affiliate links are added in the future, this policy will be updated first to describe exactly what those providers collect.

Analytics

None. We do not track visitors.

Contact

Questions about this policy: open an issue on the ClearSum repository.
